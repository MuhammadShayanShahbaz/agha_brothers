# ⚡ Agha Brothers – Electrical Parts & Solutions

A full-featured e-commerce website for industrial and commercial electrical parts, built with **Flask** (Python).  
This project includes user authentication, a shopping cart, order management, an admin dashboard, a chatbot, and email notifications.

## ✨ Features

### 🛒 Customer Features
- **User Authentication** – Register, login, and profile management.
- **Product Catalog** – Browse products by category, brand, or search.
- **Shopping Cart** – Add/update/remove items, view subtotals.
- **Order Placement** – Cash on Delivery (COD) with address validation.
- **Order History** – View past orders and order details.
- **Chatbot** – AI-powered virtual assistant for product info, prices, delivery, and more.
- **Contact Form** – Send messages directly to the store admin (saved to database + email notification).

### 🔐 Admin Features
- **Dashboard** – Overview of orders, revenue, pending orders, and low-stock alerts.
- **Order Management** – Update order status (Pending → Confirmed → Processing → Shipped → Delivered → Cancelled).
- **Product Management** – Add, edit, and deactivate products.
- **Inventory Management** – View stock levels, update stock, and see visual stock bars.
- **Customer Management** – View registered users and their order history.

### 📧 Email Notifications
- **Contact Form** – Sends an email to the admin when a user submits a message.
- **Order Confirmation** – Sends an email to the admin when a new order is placed.
